# Etch-A-Sketch

This project allows to create virtual board and draw on it. To achive that I used DOM manipulation script.

## Options

Options availeble for user are:

- change size: change number of blocks on board form 1 to 100 per side
- reset board: resets board color
- incremental saturation: block is colored with incremental saturation
- rainbow mode: block is colored with random color
- pick color: change color of the brush
